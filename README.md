Software Barbearia 💈

Sistema desktop desenvolvido em Java para gerenciamento de uma barbearia.

 📌 Sobre o projeto

O objetivo deste projeto é simular o funcionamento básico de uma barbearia, permitindo:

* Cadastro de clientes
* Login de usuários
* Agendamento de horários
* Gerenciamento de barbeiros
* Visualização de listas e informações
* Navegação por interface gráfica Java Swing

Projeto desenvolvido com foco em prática de programação orientada a objetos, interfaces gráficas e organização de sistemas desktop.

---

 🚀 Tecnologias utilizadas

* Java
* Java Swing
* Programação Orientada a Objetos (POO)
* Eclipse IDE
